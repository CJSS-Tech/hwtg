# Telegram中文资源导航

这是一个收录Telegram中文优质资源的导航网站。

## 📋 网站内容

- **中文频道**: 精选优质中文频道
- **行业交流群**: 各行业专业交流群组
- **海外商机**: 全球商业机会信息
- **求职&招聘**: 职位招聘与求职信息
- **机器人**: 实用Telegram Bot工具
- **广告业务**: 广告投放与推广服务

## 🚀 部署到 GitHub Pages

### 步骤 1: 创建 GitHub 仓库

1. 在 GitHub 上创建一个新仓库
2. 仓库名称建议使用: `telegram-navigator` 或 `username.github.io`

### 步骤 2: 上传文件

在本地项目目录中执行以下命令:

```bash
git init
git add .
git commit -m "Initial commit: Telegram中文资源导航"
git branch -M main
git remote add origin https://github.com/你的用户名/你的仓库名.git
git push -u origin main
```

### 步骤 3: 启用 GitHub Pages

1. 进入仓库的 Settings
2. 找到 Pages 选项
3. 在 Source 下选择 `main` 分支
4. 点击 Save
5. 等待几分钟后，你的网站将在 `https://你的用户名.github.io/你的仓库名/` 上线

## 📝 自定义内容

你可以编辑以下文件来自定义内容:

- `index.html` - 修改网站结构和内容
- `styles.css` - 调整样式和配色
- `script.js` - 添加或修改交互功能

## 🎨 自定义颜色

在 `styles.css` 的 `:root` 部分修改主题颜色:

```css
:root {
    --primary-color: #0088cc;  /* 主色调 */
    --secondary-color: #00aced; /* 辅助色 */
    --dark-color: #2c3e50;     /* 深色 */
}
```
